# PinHigh — Weekly Changelog

Used by the **Monday 9:00 AM** weekly report. Add a section for each week (week of Monday–Sunday). The report includes:

- **High-level overview** (this file): what was created, built, updated, and tested
- **AI competitor simulations**: count and profiles (0, 5, 10, 20 HCP, EW 2K, LPGA)
- **Trends and anomalies** from the simulation run
- **PDFs** per profile: average path, min-shots round, max-shots round

**How to use:** Add a new section `## Week of YYYY-MM-DD – YYYY-MM-DD` (Monday–Sunday). List bullet-point changes for the week.
